# Data Directory

This directory contains the JSON data files for both AWS and ONP environments.

## Structure

```
data/
├── AWS/
│   ├── api_proxys.json
│   ├── apps.json
│   ├── developers.json
│   ├── keystores.json
│   ├── products.json
│   ├── references.json
│   └── target_servers.json
└── ONP/
    ├── api_proxys.json
    ├── apps.json
    ├── developers.json
    ├── keystores.json
    ├── products.json
    ├── references.json
    └── target_servers.json
```

## Note

Copy the data files from the original ApigeeExplorer.Api/data directory to maintain the same functionality.
